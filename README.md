# 🏎️ SMASH MAN 2.0 — Multiplayer Kart Arena

![Status](https://img.shields.io/badge/Status-Beta-orange)
![Tech](https://img.shields.io/badge/Tech-Firebase%20%7C%20JS%20%7C%20CSS3-yellow)

**Smash Man 2.0** is a high-octane, browser-based multiplayer battle arena. Built with a focus on **Fast Chaos Gameplay** and **Authoritative System Design**, it leverages Firebase for real-time synchronization and Google Authentication for seamless player onboarding.

## 🧠 System Architecture & Design Pillars
This project follows a 4-pillar blueprint:
1. **Readable Combat:** Low-latency syncing ensures players understand every hit.
2. **Skill + Luck Balance:** Physics-based movement paired with randomized spawns.
3. **Reward Loop:** Integrated economy system (Kills × 2 Coins).
4. **Serverless Authority:** Real-time state management via Firebase Realtime Database.

## 🛠️ Technical Stack
- **Frontend:** HTML5, CSS3 (Custom animations & Glow effects)
- **Engine:** Vanilla JavaScript with Custom Physics/Drift Logic
- **Backend-as-a-Service:** - **Firebase Auth:** Google OAuth 2.0 integration.
    - **Firebase Realtime Database:** JSON-tree state syncing for multiplayer karts.
- **Deployment:** GitHub Pages.

## 🚀 Core Features
- [x] **Real-time Multiplayer:** Visualizing other players' karts with < 100ms latency.
- [x] **Google Authentication:** Secure persistent player profiles.
- [x] **Dynamic Movement:** Keyboard-driven velocity and position syncing.
- [ ] **Weapon System:** (Planned) Projectile synchronization and hit-registration.
- [ ] **Map Voting:** (Planned) Pre-match lobby selection system.

## 🔧 Installation & Local Development
To run this project locally:
1. Clone the repo: `git clone https://github.com/YOUR-USERNAME/smash-man-2-0.git`
2. Open `index.html` in any modern browser.
3. *Note: Ensure your local domain is added to the Firebase Authorized Domains list.*
